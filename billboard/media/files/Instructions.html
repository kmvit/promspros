UltraImageManager

Version: Extension for Opencart 2.0.1.1. But it should also work on other versions of opencart since 2.0.

Licence: Feel free to use it as you wish just don't say it's only your work. It's distributed as it is without any guarantee and you use it on your own responsibility.

This extension will save you a lot of work when creating new products in admin and assignign images to them.

UltraImageManager is able to

  • Remember the last folder in Image Manager
  • Assign multiple images to one product

Installation

  1. In your opencart admin go to Extensions -> Extension Installer
  2. Select and upload you ocmode xml file "UltraImageManager.ocmod.xml" (Note: After upload will be"Continue" button disabled. It's normal.)
  3. !IMPORTANT, DON'T FORGET: In opencart admin go to Extensions -> Modifications. Make sure that UltraImageManager is enabled and CLICK ON "REFRESH" BLUE BUTTON on the top of the page. It creates new files in virtual environment. Without this it won't work.
  4. With your ftp manager go to folder ../admin/view/javascript and replace original common.js file with new one. MAKE BACKUP of the old one.
  5. In your browser go to admin and open an edit form of any product. Press ctrl + f5 to load new common.js file into browser cache.

Usage

Select images you want to assign to the current product and click on the green button on the top of file manager. This button adds selected images to the product with the default order. That means that first image is set as default and others are set as additional images in the image tab.

When you select multiple images and you want to select different image as default then select desired images and click on the plus button under the image you want as default. The image with the clicked plus button will be set as default and the others as additional in the image tab.
